# Parking Lot - Go

## Requirements
- Multiple floors, spot types (compact, regular, large)
- Vehicle types (motorcycle, car, truck)
- Entry/exit with ticketing
- Payment by duration

## Patterns
- **Factory**: Vehicle/spot creation
- **Strategy**: Pricing strategies
- **State**: Spot availability
- **Singleton**: ParkingLot manager

## Entities
ParkingLot, Floor, Spot, Vehicle, Ticket, Payment

## Extensions
- EV charging, VIP spots, multiple entry points
